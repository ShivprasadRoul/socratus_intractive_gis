Welcome to the Climate Panchayat Data Visualization Web App

This web application allows users to access environmental index data and view GIS maps representing various environmental issues in Odisha's districts.

Explore interactive maps to visualize district-wise data for selected indexes, with color-coded maps and dynamic legends.

About the Project

This project aims to raise awareness about environmental issues through accessible data visualization.
